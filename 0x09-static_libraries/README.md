The task involves creating a static library named libmy.a which should contain all the listed functions. In case any function has not been coded, empty ones should be created with the correct prototype. The main.h file should also be pushed to the repository and should contain all the prototypes of the functions.

Another task is to create a script named create_static_lib.sh which creates a static library named liball.a from all the .c files that are present in the current directory.
