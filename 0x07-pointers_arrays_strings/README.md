s is a set of C programming tasks:

Implement a function _memset() that fills a memory area with a constant byte.
Implement a function _memcpy() that copies a memory area.
Implement a function _strchr() that locates a character in a string.
Implement a function _strspn() that gets the length of a prefix substring.
Implement a function _strpbrk() that searches a string for any of a set of bytes.
Implement a function _strstr() that locates a substring.
Implement a function print_chessboard() that prints a chessboard.
Implement a function print_diagsums() that prints the sum of the two diagonals of a square matrix of integers.
Implement a function set_string() that sets the value of a pointer to a char.
Crack the password for the crackme2 executable by creating a file that contains the password.
