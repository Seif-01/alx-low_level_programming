This is a collection of C functions that make use of recursion to solve various problems.

The first function, _puts_recursion, takes a string as input and prints it to the console followed by a newline character.
The second function, _print_rev_recursion, also takes a string as input but prints it in reverse order.
The third function, _strlen_recursion, returns the length of a given string.
The fourth function, factorial, returns the factorial of a given number. If the input is less than 0, it returns -1 to indicate an error, and if the input is 0, it returns 1.
The fifth function, _pow_recursion, returns the result of raising a given number to a given power. If the power is less than 0, it returns -1.
The sixth function, _sqrt_recursion, returns the natural square root of a given number. If the input does not have a natural square root, it returns -1.
The seventh function, is_prime_number, returns 1 if a given number is prime and 0 otherwise.
The eighth function, is_palindrome, returns 1 if a given string is a palindrome (reads the same backwards as forwards) and 0 otherwise.
The ninth and final function, wildcmp, compares two strings and returns 1 if they are identical or if the second string can be matched to the first with the special character *, which can represent any string (including an empty one).
