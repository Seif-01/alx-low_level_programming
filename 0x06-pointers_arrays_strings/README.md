Task 0: Write a function to concatenate two strings. The function should append the src string to the dest string, overwrite the terminating null byte (\0) at the end of dest, and then add a terminating null byte. Returns a pointer to the resulting string dest.

Task 1: Write a function to concatenate two strings. The _strncat function is similar to the _strcat function, except that it will use at most n bytes from src, and src does not need to be null-terminated if it contains n or more bytes. Returns a pointer to the resulting string dest.

Task 2: Write a function to copy a string. The function should work exactly like strncpy.

Task 3: Write a function to compare two strings. The function should work exactly like strcmp.

Task 4: Write a function to reverse the content of an array of integers.

Task 5: Write a function to change all lowercase letters of a string to uppercase.

Task 6: Write a function to capitalize all words of a string. The separators of words are space, tabulation, new line, ,, ;, ., !, ?, ", (, ), {, and }.

Task 7: Write a function to encode a string into 1337. Letters a and A should be replaced by 4, e and E should be replaced by 3, o and O should be replaced by 0, t and T should be replaced by 7, and l and L should be replaced by 1. You can only use one if in your code, and two loops.

Task 8: Write a function to encode a string using rot13. You can only use if statement once in your code, and two loops.

Task 9: Write a function to print an integer. You can only use _putchar function to print. You are not allowed to use long, arrays, or pointers, and hard-code special values.

Task 10: Add one line to the provided code so that it prints a[2] = 98, followed by a new line. You are not allowed to use the variable a in your new line of code or modify the variable p. You can only write one statement. Your code should be written at line 19, before the ;. Do not remove anything from the initial code.

Task 11: Write a function to add two numbers. Returns a pointer to the result. You can assume that you will always get positive numbers or 0. You can assume that there will be only digits in the strings n1 and n2. n1 and n2 will never be empty. If the result cannot be stored in r, the function must return 0.
